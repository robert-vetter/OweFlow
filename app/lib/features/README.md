# Features
Hier werden die verschiedenen Features der App organisiert.
- **schulden/**: Schuldenmanagement und Kostenaufteilung.
- **gruppen/**: Verwaltung von Gruppen und Events.
- **reminders/**: Erinnerungen für fällige Zahlungen.
- **analytics/**: Erweiterte Analysen und Visualisierungen.

Jeder Unterordner enthält:
- **models/**: Datenstrukturen.
- **screens/**: Bildschirme (UI).
- **widgets/**: Feature-spezifische Widgets.
- **controllers/**: State Management und Logik.
