# Schuldenmanagement
Hier befindet sich alles für die Verwaltung von Schulden:
- **models/**: Datenmodelle wie Schulden (debt_model.dart).
- **screens/**: Bildschirme für Schuldenübersicht und Details.
- **widgets/**: Komponenten wie Karten und Buttons.
- **controllers/**: Logik für Hinzufügen, Löschen und Verwalten von Schulden.
