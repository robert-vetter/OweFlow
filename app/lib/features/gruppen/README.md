# Gruppenverwaltung
Dieser Ordner enthält alles für die Verwaltung von Gruppen und Events:
- **models/**: Gruppenmodelle.
- **screens/**: Bildschirme für Gruppenübersicht und Details.
- **widgets/**: Komponenten wie Einladungslinks oder QR-Codes.
- **controllers/**: Logik für das Erstellen und Verwalten von Gruppen.
