# Core
Dieser Ordner enthält projektweite Ressourcen:
- **constants/**: Globale Konstanten wie Farben, Schriftarten und Abstände.
- **utils/**: Hilfsfunktionen (z. B. Formatierungen).
- **widgets/**: Wiederverwendbare Widgets wie Buttons.
- **services/**: Dienste für API-Kommunikation, OCR und Benachrichtigungen.
